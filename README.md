# msft-amzn-quant-analysis

Quant project analyzing the dynamic relationship between Microsoft and Amazon stock prices using time series methods in R.

## Contents

- `msft-amzn-quant-analysis.r`: analysis script
- `result_analysis.pdf`: project report / results

## Methods

- stationarity testing
- Box-Jenkins analysis
- Engle-Granger cointegration testing
- VAR modeling
- Granger causality
- impulse response functions
- forecast error variance decomposition
